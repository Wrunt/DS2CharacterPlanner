---Instructions---
1) unzip file to desired location (location irrelevant)
2) run "DS2 Character Planner.exe"
3) name build (optional, but useful for saving)
4) select desired class
5) enter desired stats
6) click "Update" button - will save entered values and, if necessary, set them to default min/max values

to help decide if the selected is the most soul-efficient class, click "Check Class" button

to save build, click "Save Build" button

to wipe build and start over, select "reset" button

source code available at https://github.com/Wrunt/DS2CharacterPlanner
